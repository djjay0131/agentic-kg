# Design

This folder contains design documents, architecture diagrams, and technical design decisions.

## What Goes Here

- System architecture diagrams
- Component design specifications
- API designs
- Database schemas
- Integration patterns
- Technical design decisions and trade-offs

## Document Naming Convention

Use descriptive names that indicate the feature or component:
- `feature-name-architecture.md`
- `component-name-design.md`
- `integration-xyz-design.md`
